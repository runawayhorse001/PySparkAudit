3.1.5. freq_items_df¶

PyAudit.basics.freq_items_df(df_in, top_n=3)[source]

find out the top n values and the corresponding frequency for each feature

Parameters:
  • df_in – input pandas DataFrame
  • top_n – the number of the top values
Returns:

top n values and the corresponding frequency for each feature
Author:

Wenqiang Feng and Ming Chen
Email:

von198@gmail.com

 
>>> d ={
>>>    'num': list('1223334444'),
>>>    'cat': list('wxxyyyzzzz')
>>>    }
>>> df = pd.DataFrame(d)
